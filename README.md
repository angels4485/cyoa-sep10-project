# cyoa-sep10-project
CYOA project is about creating a story using files and folders that give you different choices that you can make as you go along with the story. Have you ever played those games where you start somewhere, but then where you end up completely depends on what choices you make? Well, this is basically that, with the difference that you're in markdown files.

To start your journey as an aspiring athlete just [click here](start/start-journey.md)
